Intro to HTML

HTML stands for Hyper Text Markup Language. It is not a programming language but a Markup language,other examples include XML

Why is it not a programming language

HTML is not regarded as a programming language because you cannot define a function on it. It makes use of tags.

Python OOP recap

Python is an object oriented programming language which emphasises on the use of functions and objects An object is a collection of data(variables) and methods(functions) acts on that data while classes serves as the blueprint for the object

Main concepts of OOP

The main concepts of pyton OOP consists of Class,Objects,Polymorphism,Encapsulation and Inheritance

Other Parts of Python

Various other functions can be carried out with python like, handling of tables and documents manipulating mathematical data and much more
